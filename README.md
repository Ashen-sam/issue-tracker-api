# BugTrack â€“ Issue Tracker Web Application

BugTrack is a modern full-stack issue tracking web application designed to help teams efficiently manage bugs, issues, and project tasks. It delivers a clean, fast, and responsive user experience with real-time feedback powered by optimistic UI updates.

Built with a scalable architecture, BugTrack focuses on usability, performance, and maintainability.

---

##  Features

- Create, update, and delete issues
- Manage issue status, priority, and severity
- Optimistic UI updates for instant feedback
- Clean, minimal, and modern interface

- RESTful API architecture
